## 4. 本轮发现的问题

### 问题 1：版本差异摘要漏报“食材变化”和“步骤内容变化”

现象：

- 创建新版本时已修改食材和步骤内容
- 但 `diffSummaryText` 或 `compare.summaryText` 仍可能返回“未检测到变化”

根因：

- 后端差异计算此前只比较：
  - `ingredientsText`
  - 标签增减
  - 步骤数量
- 没有比较：
  - 食材 `rawText`
  - 步骤文案内容

处理结果：

- 已修复

涉及文件：

- [recipe-version-diff.ts](/Users/xychen/Documents/MyCode/menu-time/backend/src/server/lib/diff/recipe-version-diff.ts)
- [recipes.service.ts](/Users/xychen/Documents/MyCode/menu-time/backend/src/server/modules/recipes/recipes.service.ts)
- [recipes.mapper.ts](/Users/xychen/Documents/MyCode/menu-time/backend/src/server/modules/recipes/recipes.mapper.ts)

修复后实测结果：

- `diffSummaryText` 可返回：
  - `主料有调整；步骤内容有调整`
- `compare.summaryText` 也可返回：
  - `主料有调整；步骤内容有调整`
