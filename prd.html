# 产品需求文档（PRD） ## 家庭菜谱手账小程序 · 食光记 > 版本：V1.0 MVP > 日期：2026-03-23 > 目标：单人版跑通核心闭环，支持后期扩展至多成员家庭 ## 1. 产品概述 ### 1.1 定位 一个面向家庭菜谱管理的轻量级手账工具，以菜谱版本管理和时间轴回忆为核心差异点，解决“同一道菜多次改良后忘记最佳配方”和“记录餐桌美好时刻”的痛点。 ### 1.2 核心价值主张 - **版本可控**：一道菜的 N 种做法完整留存，差异一目了然。 - **回忆可溯**：每道菜的历史时刻按时间聚合，点开即见过往故事。 - **决策减负**：随机点菜解决选择困难，计划生成简化备餐流程。 ### 1.3 用户画像 | 角色 | 描述 | 当前阶段 | | --- | --- | --- | | 管理员 | 家庭菜谱主导者，录入、编辑、计划菜单 | MVP 唯一角色 | | 成员 | 查看、点菜、评论、打分 | 二期开放 | ## 2. 功能架构 ### 2.1 信息架构 ```text 首页 ├─ 本周菜单快捷入口 ├─ 最新时光动态 └─ 随机点菜快捷入口 菜谱库 ├─ 分类筛选 ├─ 标签筛选 └─ 菜谱卡片（显示版本数、最新记录日期） 菜谱详情 ├─ 当前版本（做法） ├─ 版本历史（对比） ├─ 时光轴（按菜谱聚合） └─ 操作：记一笔 / 新建版本 / 加入点菜 点菜台 ├─ 本周菜单（周历视图） ├─ 随机点菜（选择困难模式） └─ 生成购物清单 我的 ├─ 分类管理 ├─ 标签管理 └─ 设置 ``` ### 2.2 功能清单 | 阶段 | 模块 | 功能 | 优先级 | | --- | --- | --- | --- | | MVP | 菜谱管理 | 创建菜谱（名称、分类、标签、主料、步骤、小贴士） | P0 | | MVP | 菜谱管理 | 版本管理：新建版本、版本对比 | P0 | | MVP | 菜谱管理 | 分类与标签自定义 | P0 | | MVP | 时光记录 | 绑定菜谱创建时刻（文字、照片、日期、参与人） | P0 | | MVP | 时光记录 | 打分：好吃程度、难度 | P0 | | MVP | 点菜计划 | 手动添加菜品至本周菜单 | P0 | | MVP | 点菜计划 | 随机点菜（筛选 + 抽取 + 动画） | P0 | | MVP | 点菜计划 | 生成购物清单（菜品 / 调料分离） | P0 | | 二期 | 家庭协作 | 邀请成员、角色权限 | P1 | | 二期 | 推荐优化 | 基于历史的智能推荐 | P1 | | 二期 | 时间轴升级 | 首页全局时间轴 | P1 | | 三期 | 实体手账 | 数据导出供排版印刷 | P2 | | 三期 | 语音录入 | 步骤语音转文字 | P2 | ## 3. 核心功能详解 ### 3.1 菜谱录入 **入口** - 首页“+”或 菜谱库“新建”。 **录入字段** - 菜谱名称（必填） - 分类（选填，下拉选择，支持新建） - 标签（选填，多选，支持新建） - 主料（选填，文本） - 步骤（选填，多条文本） - 小贴士（选填，文本） **保存后** - 创建 V1 版本 - 询问是否记录本次做菜时刻 ### 3.2 版本管理 **新建版本场景** - 同一道菜做法改良、换主料、调整口味。 **流程** - 复制上一版本内容作为草稿 - 修改所需字段 - 系统自动生成差异摘要（主料变化、标签变化、步骤数变化） - 保存为新版本，成为当前默认版本 **版本对比** - 版本列表展示所有历史版本 - 点击任一版本，显示与当前版本的差异摘要 - 差异类型：主料调整、标签新增 / 移除、步骤数变化 ### 3.3 时光记录 **触发时机** - 保存新版本时，或随时手动添加 **记录内容** - 关联菜谱（默认当前，可切换） - 日期（默认今天，可修改历史日期） - 文字内容（故事、反馈、备注） - 照片（最多 9 张） - 参与人（文本标签，如“全家”“小明生日”） - 打分（可选）：好吃程度 1-5 星、难度 1-5 星 **展示方式** - 按菜谱聚合，时光轴形式呈现 - 最新记录置顶 - 每条记录显示：日期、内容预览、照片缩略图、打分 ### 3.4 点菜计划 **手动点菜** - 周历视图（周一到周日） - 点击时段添加菜品 - 支持搜索菜谱库 - 支持选择具体版本 - 拖拽调整顺序 **随机点菜（选择困难模式）** **筛选条件（可选）** - 分类（多选） - 标签（多选） - 难度上限 - 排除最近 N 天吃过的菜 - 特定成员偏好标签 **抽取过程** - 点击开始，进入全屏动画 - 菜谱卡片快速切换，逐渐减速停止 - 震动反馈，结果弹性呈现 **结果操作** - 就它了：加入点菜台 - 再来一次：同条件重新抽取 - 换版本：切换该菜的其他版本 - 类似推荐：同分类 / 标签的其他菜 - 查看详情：跳转菜谱页 **连抽模式** - 一次生成整周 7 道菜 - 自动去重 - 简单均衡分类（避免全肉或全素） **抽取历史** - 本次随机 session 的抽取记录 - 防止手快错过想选的菜 ### 3.5 购物清单 **生成逻辑** - 解析本周菜单所有菜品 - 关键词识别区分菜品与调料 - 自动去重 - 分为两栏：菜品清单、调料清单 **清单操作** - 手动勾选已购 - 编辑数量备注 - 复制文本 - 生成图片分享 ## 4. 角色与权限 | 功能 | 管理员 | 成员（二期） | | --- | --- | --- | | 创建 / 编辑 / 删除菜谱 | ✅ | ❌ | | 创建新版本 | ✅ | ❌ | | 查看所有版本 | ✅ | ✅ | | 记录时光 | ✅ | ✅ | | 点菜 | ✅ | ✅ | | 生成 / 编辑购物清单 | ✅ | 仅查看 | | 管理分类标签 | ✅ | ❌ | | 邀请 / 管理成员 | ✅ | ❌ | ## 5. 关键页面与交互 ### 5.1 首页 - 顶部：本周菜单快捷入口（显示“本周已规划 X 道菜”）。 - 主体：最新时光记录流（仅自己的记录，按时间倒序）。 - 卡片：菜谱封面图（或首图）+ 菜谱名称 + 日期 + 一句话预览。 - 点击 → 进入该菜谱详情页并定位到对应时光记录。 ### 5.2 菜谱库 - 顶部：分类 Tab（全部 / 肉菜 / 素菜 / 汤 / 点心）+ 标签筛选器。 - 列表：卡片网格 / 列表切换。 - 卡片：封面图、菜名、当前版本标签（如“V2 菠萝版”）、最近记录日期。 - 角标：时光记录数量（如有）。 - 搜索：按菜名搜索。 ### 5.3 菜谱详情 - 顶部：大图、菜名、分类标签、当前版本标识 - 操作栏：记一笔 / 新建版本 / 加入点菜 - Tab 切换： - 做法（当前版本内容） - 版本（列表 + 对比） - 时光轴（历史记录） ### 5.4 版本对比页 - 当前版本 vs 选中版本 - 差异摘要卡片（系统自动生成） - 下方展开两个版本完整内容 ### 5.5 点菜台 - 随机点菜入口 - 周历视图，每餐一个卡片槽 - 已选菜品显示缩略图 + 菜名 - 长按删除或调整顺序 - 底部：“生成购物清单”按钮 ### 5.6 随机点菜页 - 筛选面板（可折叠） - 快速场景标签（一键设置） - 大按钮：开始随机 - 结果页：动画效果、菜谱信息、操作按钮、本次抽取记录 ### 5.7 购物清单页 - 分栏 Tab：菜品 / 调料 - 清单项：名称、勾选框、数量备注（可编辑） - 底部操作：全选 / 反选、清空已购、复制文本、生成图片 ## 6. 业务规则 ### 6.1 版本管理规则 - 每个菜谱至少有一个版本（V1） - 版本号自动递增，支持自定义版本名称（如“菠萝版”） - 新建版本默认复制上一版本内容 - 当前版本为最新版本，可随时切换历史版本为当前 ### 6.2 时光记录规则 - 必须关联菜谱，可选关联具体版本 - 日期可回溯，支持补记 - 同菜谱多条记录按时间倒序排列 - 最新记录的图片作为菜谱封面（如无自定义封面） ### 6.3 随机点菜规则 - 候选池排除：本周已吃、本次 session 已跳过、用户标记“最近不想吃” - 连抽 7 天自动均衡分类（至少 1 汤 1 素） - 筛选结果为空时，提示放宽条件 ### 6.4 购物清单规则 - 调料识别关键词：酱、油、盐、糖、醋、料酒、葱、姜、蒜、胡椒、淀粉、鸡精、味精、蚝油、生抽、老抽、香油、花椒、八角、桂皮等 - 其余归为原料 / 菜品 - 去重逻辑：相同名称合并为一项 ## 7. 可能的扩展项 ### 7.1 功能扩展 | 方向 | 描述 | | --- | --- | | 食材反向匹配 | 输入现有食材，推荐可做的菜 | | 节气 / 节日模式 | 春节、中秋等节日优先推荐往年高分菜 | | 营养分析 | 记录热量、蛋白质等，生成周报 | | cooking 模式 | 步骤全屏显示，支持语音切换下一步 | | 视频记录 | 时光记录支持短视频 | | 多人协作编辑 | 多人共同维护同一道菜谱 | | 菜谱传承 | 长辈菜谱授权给子女独立管理 | | 社区发现 | 公开优质家庭菜谱，发现同口味家庭 | ### 7.2 商业模式扩展 | 方向 | 描述 | | --- | --- | | 实体手账定制 | 按季度 / 年度生成精美排版 PDF，支持印刷 | | 食材电商 | 购物清单一键下单买菜 | | 会员服务 | 高级功能（无限云存储、专属模板、优先客服） | | 企业团餐 | 小型团队 / 公司午餐计划管理 | ### 7.3 平台扩展 | 方向 | 描述 | | --- | --- | | 独立 App | 脱离微信，更深度系统整合 | | 智能设备 | 与智能冰箱联动（识别库存食材推荐菜谱） | | 车载场景 | 通勤路上语音规划晚餐菜单 | ## 8. 成功指标 | 指标 | 目标 | 说明 | | --- | --- | --- | | 菜谱创建数 | ≥20 道 | 验证录入体验 | | 版本创建率 | ≥30% | 验证版本管理价值 | | 时光记录数 | ≥10 条 | 验证回忆功能 | | 随机功能使用率 | ≥50% 点菜行为 | 验证核心差异点 | | 周菜单完成度 | ≥3 次 / 周 | 验证闭环跑通 |
